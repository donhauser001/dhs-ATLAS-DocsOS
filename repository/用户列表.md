# 用户管理 {#user-index}

```yaml
type: directory_index
id: user-list
title: 系统用户
status: active

source:
  function: principal
  filter:
    status: active

display:
  fields:
    - { path: display_name, label: 姓名 }
    - { path: "identity.emails[0]", label: 邮箱 }
    - { path: status, label: 状态, type: badge }
  views:
    - { type: card, default: true, columns: 3 }
    - { type: table }
  pagination:
    page_size: 20

interaction:
  click: open_document
  search:
    enabled: true
    fields: [display_name, identity.emails]

$display:
  layout: card_list
  color: { token: color.brand.primary }
  icon: { token: icon.general.users }
```

本文档为系统用户的汇总入口。

## 功能说明

- **自动发现**: 系统自动扫描所有 `atlas.function: principal` 的文档
- **卡片/表格视图**: 支持切换显示模式
- **搜索功能**: 可按姓名、邮箱搜索用户
- **点击跳转**: 点击用户可直接进入用户详情页

## 新增用户

创建新用户只需：
1. 在 `users/principals/` 目录下创建 `u-xxx.md`
2. 添加 `atlas.function: principal` 声明
3. 系统会自动将新用户加入列表
