## 视图说明

### 月视图 (calendar.month)
- 传统月历视图
- 显示整月的日程安排
- 支持点击日期查看详情

### 周视图 (calendar.week)
- 按周展示日程
- 显示具体时间段
- 适合详细时间安排

### 热力图 (calendar.heatmap)
- 类似 GitHub 贡献图
- 用颜色深浅表示活跃度
- 适合展示工作量统计
