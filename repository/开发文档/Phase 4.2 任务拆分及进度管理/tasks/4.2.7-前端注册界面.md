## 技术要点

### 注册流程

```
1. 用户填写表单（用户名、邮箱、手机、密码）
2. 前端验证
3. 提交到后端
4. 后端创建用户文档（写入 user_document_directory）
5. 触发索引更新
6. 发送激活邮件（如果需要）
7. 返回结果，跳转到激活页面或登录页面
```

### 表单字段

| 字段 | 必填 | 验证规则 |
|------|------|----------|
| 用户名 | ✓ | 字母开头，3-32 字符 |
| 邮箱 | ✓ | 邮箱格式，唯一 |
| 手机 | - | 手机格式，唯一 |
| 密码 | ✓ | 根据系统密码策略 |
| 确认密码 | ✓ | 与密码一致 |

### 密码强度

```typescript
// 密码强度计算

```
