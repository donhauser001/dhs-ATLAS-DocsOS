## 文档说明

本文档是 ATLAS 系统的第一份 ADL Facts 文档，用于验证 Phase 0 的核心闭环：

1. **Parser** 能正确解析所有 Block
2. **Renderer** 能正确投射为 UI
3. **Proposal** 能正确生成变更提案
4. **Validator** 能正确校验变更
5. **Executor** 能正确执行并生成 Git Commit

### Phase 2.5 语义化改进

本文档已升级为语义引用写法：

- 分类使用 `$display` 命名空间定义颜色和图标
- 服务使用 `{ ref: "#anchor" }` 引用分类
- 颜色和图标从 `genesis/tokens.md` 的 Token 系统获取

这样做的好处：
- 渲染器不再硬编码颜色映射
- 新增分类无需修改代码
- 视觉一致性由 Token 系统统一管理
